# Phishing Email Analysis

## Introduction

This project shows how to analyze a phishing email and identify suspicious activity.

## Objective

The main objective of this project is to understand how phishing emails work and how they can be detected.

## Components Analyzed

* Email Header
* Sender Information
* SPF and DKIM Authentication
* Suspicious Links
* Phishing Techniques

## Conclusion

This project helps in understanding email security and how to detect phishing emails.
